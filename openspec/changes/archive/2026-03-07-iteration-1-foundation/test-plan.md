## 二、新建项目流程

### T2.1 新建项目对话框
1. 点击 "New Project" 按钮
2. **预期**: 弹出模态对话框，包含 "Project Name" 和 "Storage Path" 两个输入框
3. 不填任何内容，直接点击 "Create"
4. **预期**: 表单校验失败，提示 "Project name is required" 和 "Please select a storage path"

### T2.2 项目名校验
1. 输入超过 64 个字符的项目名
2. **预期**: 提示 "Max 64 characters"
3. 输入含有特殊字符（如 `<>:"/\|?*`）的名称
4. **预期**: 提示 "Contains invalid characters"
5. 输入合法名称（如 "My Circuit"）
6. **预期**: 校验通过，无错误提示

### T2.3 路径选择
1. 点击 Storage Path 输入框右侧的文件夹图标
2. **预期**: 弹出系统文件夹选择对话框
3. 选择一个文件夹后确认
4. **预期**: 路径正确回填到输入框中

### T2.4 创建项目并进入编辑器
1. 填写合法的项目名和路径，点击 "Create"
2. **预期**: 对话框关闭，页面从欢迎页切换到编辑器视图
3. **预期**: 编辑器界面显示完整的布局（工具栏、画布、左右面板、Sheet标签页、状态栏）

> **后续版本功能**: 创建项目时在磁盘上生成实际文件（project.json 等）→ Iteration 3
> 当前版本仅切换 UI 视图，不创建任何文件。

### T2.5 打开项目
1. 回到欢迎页（菜单 File → New Project，或 Ctrl+N）
2. 点击 "Open Project" 按钮
3. **预期**: 弹出系统文件夹选择对话框
4. 选择任意文件夹并确认
5. **预期**: 进入编辑器视图

> **后续版本功能**: 打开项目时读取并校验项目文件 → Iteration 3
> 当前版本选择任意文件夹即可进入编辑器，不验证内容。

## 三、编辑器布局

### T3.1 整体布局结构
1. 进入编辑器后检查各区域
2. **预期**: 从上到下依次为 —— 工具栏 | 左面板+画布+右面板 | Sheet标签页 | 状态栏

1. **预期**: 左侧面板显示元器件库，宽度约 240px
2. **预期**: 顶部有搜索框
3. **预期**: 分类列表展示，有器件的分类默认展开：无源器件、二极管、常用IC、电源符号
4. **预期**: 无器件的分类显示 "Coming soon"：三极管、MOS管、运算放大器、逻辑门、连接器、其他

### T3.3 右面板 — AI 助手
1. **预期**: 右侧面板显示 "AI Assistant" 标题
2. **预期**: 中间显示占位文字 "AI Schematic Assistant"
3. **预期**: 底部输入框和发送按钮处于**禁用状态**
4. **预期**: "AI Analysis" 和 "Clear" 按钮处于禁用状态

> **后续版本功能**: AI 对话交互、流式输出、原理图生成 → Iteration 2

### T3.4 面板收起/展开

2. **预期**: 面板收起时平滑过渡（约 200ms），画布区域自动扩展

4. 通过菜单 View → AI Chat Panel (Ctrl+Shift+R) 切换右面板

### T3.5 面板宽度调节

4. **预期**: 宽度可在 180px ~ 360px 范围内调节
5. 对右面板执行同样操作

### T3.6 状态栏
1. **预期**: 状态栏显示 X/Y 坐标、缩放百分比、Grid 状态
2. **预期**: 右侧显示 "● Offline"（AI 未连接状态）

4. **预期**: X/Y 坐标实时更新

### T3.7 Sheet 标签页

## 六、元器件操作

### T6.1 拖拽放置元器件
1. 在左面板元器件库中，鼠标按住 "电阻" 项
2. 拖拽到画布区域释放
3. **预期**: 画布上出现电阻符号，自动分配编号（R1）
4. 再拖一个电阻
5. **预期**: 编号为 R2（递增，不复用）
6. 对电容（C1, C2...）、二极管（D1, D2...）、LED（LED1...）分别测试

### T6.2 双击放置元器件
1. 在元器件库中双击 "电容"
2. **预期**: 画布中心位置出现一个电容符号

### T6.3 IC 元器件
1. 拖拽 "8引脚IC" 到画布
2. **预期**: 出现矩形 IC 符号，左右各 4 个引脚，引脚有编号标注
3. 拖拽 "16引脚IC" 到画布
4. **预期**: 更大的 IC 矩形，左右各 8 个引脚

### T6.4 电源符号
1. 拖拽 "VCC" 到画布
2. **预期**: 显示红色向上箭头符号，带一个端口
3. 拖拽 "GND" 到画布
4. **预期**: 显示黑色三线接地符号，带一个端口

### T6.5 元器件选中
1. 单击一个元器件
2. **预期**: 出现选中框（蓝色虚线 + 控制点）
3. 单击画布空白区域
4. **预期**: 取消选中

### T6.6 元器件删除
1. 选中一个元器件，按 Delete 键
2. **预期**: 元器件被删除

4. **预期**: 同样被删除

### T6.7 复制粘贴
1. 选中一个元器件（如 R1），按 Ctrl+C
2. 按 Ctrl+V
3. **预期**: 在原位偏移 (20, 20)px 处出现副本，自动分配新编号（如 R3，而非 R1 的副本）
4. 连续多次 Ctrl+V
5. **预期**: 每次都生成新编号且偏移放置

### T6.8 元器件移动
1. 选中一个元器件并拖拽移动
2. **预期**: 元器件跟随鼠标移动
